312 def ISO8601():
313 """Return the object in ISO 8601-compatible format containing
314 the date, time with seconds-precision and the time zone
315 identifier - see http://www.w3.org/TR/NOTE-datetime
316  
317 Dates are output as: YYYY-MM-DDTHH:MM:SSTZD
318 T is a literal character.
319 TZD is Time Zone Designator, format +HH:MM or -HH:MM
320  
321 The HTML4 method below offers the same formatting, but
322 converts to UTC before returning the value and sets the TZD"Z"
323 """
324  
325 def HTML4():
326 """Return the object in the format used in the HTML4.0
327 specification, one of the standard forms in ISO8601. See
328 http://www.w3.org/TR/NOTE-datetime
329  
330 Dates are output as: YYYY-MM-DDTHH:MM:SSZ
331 T, Z are literal characters.
332 The time is in UTC.
333 """
